# README

Sytem Requirements:
- Ruby 3.0.1
- Rails 6.1.3.2
- Bundler 2.2.15
- PostgreSQL 13.2
- Node 14.17.0
- Yarn 1.22.10

After cloning, go to the project folder and run:
- bundle install
- yarn install
- rails db:create
- rails db:migrate

run "rails s" to start the server and go to http://localhost:3000 to see the app running.
